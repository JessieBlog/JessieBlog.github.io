循环语句

发表于 2017-06-04

while循环

While循环语句可以有条件地将内嵌语句执行0遍或者若干遍

while（boolean_expression）

{  
       embeded-statement;  //满足布尔表达式时执行，否则停止循环
}

While循环语句执行时，先判断while后面括号内的语句的值，当为False时，不执行大括号内的嵌入程序段，若为True 时，进入循环，执行循环内程序段一遍后，再次判断条件是否满足，若满足就一直执行下去，一直到不满足为止，跳出循环，继续后面的语句。

do…while循环

程序中使用循环时，第一次不检查条件是否满足，直接进入循环，第二次以后才检查条件是否满足，条件为True时，才能进入循环。

do  
{  
      embeded-statement; //直接执行 
}   
while(boolean expression)； //满足布尔表达式时进入循环，否则停止循环

for循环

在程序的设计过程中，若希望从某个值开始，每执行指定的程序段一次，便将该数值增加（减少）一个单位，如果结果始终比终值小（大），便继续执行该程序段，一直到不满足终值才离开该程序段。这时，我们就可以使用For循环语句。

For循环语句的基本语法：

for（initializer;condition;iterator） 
{  
   statement;  //循环语句
}

其中Initializer，Condition，Iterator这3个项都是可选项。Initializer为循环控制变量作初始化，循环控制变量可以有一个或多个；Condition为循环控制条件，也可以有一个或者多个语句；Iterator为按规律改变循环控制变量的值。

跳过或中断循环

C#里面的循环代码,有时候,因为某些需要,要对某些条件的循环进行跳过或者中断循环,continue就是用来跳过循环的,break是用来中断循环的。

使用方法：

continue和break的用法一样,直接写上这个单词,后面加一个分号就行

比如:
continue;
break;

实例

当i==3的时候,执行continue;程序直接跳回for里面执行,而 Console.Write(i);在本轮循环中不再执行,实际上就是跳过去了。

从i等于4开始的循环被中止了,只输出到3就不再往下输出
