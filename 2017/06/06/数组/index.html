title:数组

数组是一个存储相同类型元素的固定大小的顺序集合。

声明数组

在C#中声明一个数组使用以下语法：

dataype[] arrayName;
  • dataype用于指定被存储在数组中的元素的类型。
  • []指定数组的秩（维度）。秩指定数组的大小。
  • arrayName指定数组的名称。

例如：

string[] playerID;

初始化数组

声明一个数组不会在内存中初始化数组。当初始化数组变量时，可以赋值给数组。

数组是一个引用类型，需要使用new关键字创建数组的实例。

例如：

string[] playerID = new.string[10];

赋值给数组

1.可以通过使用索引号赋值给一个单独的数组元素

例如：

string[] playerID = new string[10];
string[0] = "qiu";

2.可以在声明数组的同时给数组赋值

例如：

string[] playerID = {"qiu","fei","yang","shao"};

3.可以创建并初始化一个数组

例如：

char[] m_char = new char[5] {'1','2,'3','4','5'};

4.在上述情况下，可以省略数组大小

例如：

char[] m_char = new char[] {'1','2,'3','4','5'};

5.也可以赋值一个数组变量到另一个目标数组变量中。在这种情况下，目标和源会指向相同的内存位置：

int[] marks = new int[] {99,98,99,97,92};
int[] score = marks;

当您创建一个数组时，C# 编译器会根据数组类型隐式初始化每个数组元素为一个默认值。例如，int 数组的所有元素都会被初始化为 0。

访问数组元素

元素是通过带索引的数组名称来访问的。这是通过把元素的索引放置在数组名称后的方括号中来实现的。

例如：

int[] score0 = marks[2] ;
