数据类型

数据类型包括直类型和引用类型。

值类型：简单类型，结构类型，枚举类型。

引用类型：object（对象）

简单类型：int（整型）、double（单精度浮点型）、flaot（双精度浮点型）、bool（布尔值）、buyt（字节）、char（字符）、datetime（时间）

string在.net中是一个比较特殊的类型，它表现出来是值类型，但是准确的讲它应该是"内置的引用类型"
  • 拓展

整型（INTEGER）数据是不包含小数部分的数值型数据，用字母I表示。整型数据只用来表示整数，以二进制形式存储。

char: 字符型数据，属于整形数据的一种，占用一个字节

unsigned char:无符号 字符型数据，属于整形数据的一种，占用一个字节

short:短整形数据，属于整形数据的一种，占用两个字节

unsigned short:无符号短整型数据，属于整形数据的一种，占用两个字节

int:整形数据，属于整形数据的一种，占用两个 字节

unsigned int:无符号整型数据，属于整形数据的一种，占用两个字节

long: 长整型数据，属于整形 数据的一种，占用四个字节

unsigned long:无符号 长整型数据，属于整形数据的一种，占用四个字节

数据类型转换

一、隐式转换：自动类型转换

方法一：使用 Parse() 方法进行转换

1）、字符串转换为整型用int.Pares()

string str="13";
 int number=int.Pares(str);

2）、字符串转换为双精度浮点型用double.Pares() 

string str="14";
 double number =double.Pares(str);

3）、字符串转换为单精度浮点型用flaot.Pares() 

string str="1314"
flaot number= flaot.Pares(str);

方法二：使用Convert类进行转换(object value)

1、数值型转换为字符型

例如：整型转换为字符型
int a=100;
string b=Convert.ToString(a);

2、字符串转换为数值型

例如：字符串转换为整型
string a="123456";
int b=Convert.ToInt32(a);
字符串转换为单晶度浮点型
string a="123456";
flaot b=Convert.ToSingle(a);

二、显式转换：强制类型转换

例如：
double a=12.2;
int b=10;
int c=(int)a+b;
