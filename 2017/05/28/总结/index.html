程序

  • 程序：人们根据自己的需求完成任务的过程。

  • 计算机程序：计算机根据人们的需求完成任务的指令集。

  • 代码转换过程：C++、C#、Java等高级语言 → 汇编语言 → 二进制机器指令。

  • 程序运行的原理：无论多高级的语言编写的程序，最终都是转化为机器语言（0110）运行的。

计算机硬件组成

计算器硬件五大组成部件：运算器、控制器、存储器、输入设备、输出设备

  1. 运算器：中央处理器（CPU）

  2. 控制器：计算机的指挥系统

  3. 存储器：内存和外存

  4. 输入设备：键盘、鼠标

  5. 输出设备：显示器
